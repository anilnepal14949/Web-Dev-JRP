PHP Filters

PHP provides filter_var() function to sanitize and validate data.

Syntax: filter_var(variable, filter, options)

Sanitizing a String

FILTER_SANITIZE_STRING filter is used for sanitizing string.

Example: filter_var($string, FILTER_SANITIZE_STRING)

Validate an Integer

FILTER_VALIDATE_INT filter is used for validating integers.

Example: filter_var($int, FILTER_VALIDATE_INT)

Validate IP Addresses

Validate Integers Within a Range

We can pass options to filter_var() function to validate for any ranges.

Example: filter_var($int, FILTER_VALIDATE_INT, array("options"=>array("min_range"=>0, "max_range"=>100)))

Validate IP Addresses

FILTER_VALIDATE_IP filter is used for validating IP Addresses.

Example: filter_var($ip, FILTER_VALIDATE_IP)

Sanitizing and Validating Emails

FILTER_SANITIZE_EMAIL and FILTER_VALIDATE_EMAIL filters are used for sanitizing and validating emails respectively.

Example: filter_var($email, FILTER_SANITIZE_EMAIL); filter_var($email, FILTER_VALIDATE_EMAIL)

Sanitizing and Validating URLS

FILTER_SANITIZE_URL and FITLER_VALIDATE_URL filters are used for sanitizing and validating URLs.

Example: filter_var($url, FILTER_SANITIZE_URL); filter_var($url, FILTER_VALIDATE_URL);

If there is need to check if the URL has any query string or not, use FILTER_FLAG_QUERY_REQUIRED like below:

filter_var($url, FILTER_VALIDATE_URL, FILTER_FLAG_QUERY_REQUIRED);
