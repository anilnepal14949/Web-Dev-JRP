CSS Table

CSS table properties are used to style the table borders, size, alignment, style and make it responsive.

CSS Table Properties:

  • width
  • height
  • border
  • border-collapse
  • vertical-align
  • caption-side
  • empty-cells

border-collapse property:

<table style="border-collapse: separate;"> <tr> <td> Cell A </td> </tr> <tr> <td> Cell B </td> </tr> </table>

Output (separate borders):

Cell A
Cell B
<table style="border-collapse: collapse;"> <tr> <td style="border: 3px solid #000;"> Cell A </td> </tr> <tr> <td style="border: 5px dotted #f00;"> Cell B </td> </tr> </table>

Output (collapsed borders):

Cell A
Cell B

vertical-align property

To set the horizontal alignment, text-align property can be used. However, if the content needs vertical alignment, vertical-align property can be used.

<table style="border-collapse:collapse"> <tr> <td style="height: 40px; border: 2px solid #f00; vertical-align: bottom;"> Cell A </td> </tr> <tr> <td style="height: 40px; border: 2px solid #f00; vertical-align: top;"> Cell B </td> </tr> </table>

Output:

Cell A
Cell B

caption-side property

caption-side property can be used to set the alignment for the caption.

<table style="border-collapse:collapse; width: 400px;"> <caption style="caption-side: bottom"> This caption is set in the bottom of the table </caption> <tr> <td style="height: 40px; border: 2px solid #f00; vertical-align: bottom;"> Cell A </td> </tr> <tr> <td style="height: 40px; border: 2px solid #f00; vertical-align: top;"> Cell B </td> </tr> </table>

Output:

This caption is set in the bottom of the table
Cell A
Cell B

empty-cells property

empty-cells property can be used to determine whether the border should be displayed in empty cells or not.

<table style="border-collapse:collapse; width: 200px; empty-cells: hide;"> <tr> <td style="height: 40px; border: 2px solid #f00;"> Cell A </td> <td style="height: 40px; border: 2px solid #f00;"> Cell B </td> </tr> <tr> <td style="height: 40px; border: 2px solid #f00;"> Cell C </td> <td style="height: 40px; border: 2px solid #f00;"></td> </tr> </table>

Output:

Cell A Cell B
Cell C

CSS Links

CSS link properties are used to set the unvisited, visited, hover and active states of a link.

CSS Link States

Different states of a link can be used to style links in CSS; also known as pseudo-classes. The order defined in the list below should be followed while defining the pseudo classes for effectiveness.

  • :link : Used to style the links
  • :visited : Used to style the visited links
  • :hover : Used to style the links on hover
  • :active : Used to style the links when clicked
While styling these pseudo classes, the text, fonts, color, background and all other properties can be used as per requirements.

CSS Display/Visibility

CSS display property can be used to specify how an element should be displayed.

Values:

  • block
  • inline
  • inline-block
  • none

Examples:

<p style="display:inline;> This is first line </p> <p style="display:inline;"> This is second line </p>

Output:

This is first line

This is second line

CSS Visibility:

CSS visibility property can be used to make any element visible or hidden.

Example:

<div style="width: 100px; height: 100px; background-color: #f00; color: #fff; visibility: visible"> This is visible </div> <div style="width: 100px; height: 100px; background-color: #f00; color: #fff; visibility: hidden"> This is hidden </div> <div style="width: 100px; height: 100px; background-color: #f00; color: #fff; visibility: visible"> This is visible again </div>

Output:

This is visible
This is hidden
This is visible again

How is display:none different from visibility:hidden?

The property display:none removes the element from the webpage while the visibility:hidden only hides the element from the webpage while the space covered by the element is still visible.

Example:

<div style="width: 100px; height: 100px; background-color: #f00; color: #fff;"> This is visible </div> <div style="width: 100px; height: 100px; background-color: #f00; color: #fff; display: none"> This is hidden </div> <div style="width: 100px; height: 100px; background-color: #f00; color: #fff;"> This is visible again </div>

Output:

This is visible
This is hidden
This is visible again

CSS Float

CSS float property is used to position different elements.

Values:

  • left
  • right
  • none

clear Property

The clear property is used to clear the effect caused by float property.

Values:

  • none
  • left
  • right
  • both

    • Example of float property:

    Without Float:

    <div style="width:200px;height:200px;border:4px solid #000;"> <p> This is first div </p> </div> <div style="width:200px;height:200px;border:4px solid #888;"> <p> This is second div </p> </div>

    Output:

    This is first div

    This is second div

    With Float:

    <div style="float:left;width:200px;height:200px;border:4px solid #000;"> <p> This is first div </p> </div> <div style="float:right;width:200px;height:200px;border:4px solid #888;"> <p> This is second div </p> </div>

    Output:

    This is first div

    This is second div

    Float next to each other:

    <div style="float:left;width:200px;height:200px;border:4px solid #000;"> <p> This is first div </p> </div> <div style="float:left;width:200px;height:200px;border:4px solid #888;"> <p> This is second div </p> </div>

    Output:

    This is first div

    This is second div

    CSS Positioning

    CSS position property can be used to place element in any location within the page.

    There are two types of positioning in CSS:

    • Relative
    • Absolute

    Relative Positioning

    The position:relative property is used to change the position of the HTML element from the position it is displayed.

    Absolute Positioning

    The position:absolute property is used to change the position of the HTML element from the position relative to its parent. If no parent specified, the positioning starts from top-left corner on the screen.

    Values:

    • left
    • right
    • top
    • bottom

    CSS z-index

    CSS z-index property can be used to stack elements on top of each other.

    z-index property only works on elements with position property set or with flex elements.

    Example:

    <div style="position: relative; width:100px; height:100px; background:#132467;"></div> <div style="position: relative; top: -40px; width:100px; height:100px; background:#525478;"></div>

    Without z-index: The second div has top -40px. So, that should be stacked on top of the first div.

    <div style="position: relative; z-index:1; width:100px; height:100px; background:#132467;"></div> <div style="position: relative; top: -40px; width:100px; height:100px; background:#525478;"></div>

    CSS Overflow

    CSS overflow property display scroll bar if the content of the element is larger than the element dimensions. Overflow can be used separately for horizontal and vertical contents using overflow-x and overflow-y respectively.

    Values:

    • visible
    • hidden
    • scroll
    • auto

    Examples:

    <div style="width:200px;height:100px;border:2px solid #000;overflow:visible;"> <p> This is a visible text, more visible text, more visible text, more visible text and more visible text. </p> </div>

    Output:

    This is a visible text, more visible text, more visible text, more visible text and more visible text.

    <div style="width:200px;height:100px;border:2px solid #000;overflow:hidden;"> <p> This is a visible text, more visible text, more visible text, more visible text and more visible text. </p> </div>

    Output:

    This is a visible text, more visible text, more visible text, more visible text and more visible text.

    <div style="width:200px;height:100px;border:2px solid #000;overflow:scroll;"> <p> This is a visible text, more visible text, more visible text, more visible text and more visible text. </p> </div>

    Output:

    This is a visible text, more visible text, more visible text, more visible text and more visible text.

    <div style="width:200px;height:100px;border:2px solid #000;overflow:auto;"> <p> This is a visible text, more visible text, more visible text, more visible text and more visible text. </p> </div>

    Output:

    This is a visible text, more visible text, more visible text, more visible text and more visible text.
