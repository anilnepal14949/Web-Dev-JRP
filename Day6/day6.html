CSS Syntax

A CSS style is made of three parts:

  • Selector: A selector could be any HTML element that is to be styled.
  • Property: A property refers to any type of HTML attribute.
  • Value: Values are similar to the values of attributes. Each value is assigned to a property.

Example: table { background-color: yellow; }

In the example above, table is the selector, background-color is the property and yellow is the value of the background-color property.

The above example is similar to <table bgcolor="yellow">

Types of Selectors

Selectors can be more than just html tags. Below are the various types of selectors used in selecting elements in CSS

Type Selectors

Type selector is used to select HTML tags as shown in the example above.

Example: h1 { color: blue; }

In the example above h1 is a type selector as h1 is an HTML tag

Universal Selectors

If there is a need to specify some styles for all the elements in a page, a universal selector can be used

Example: * { color: black; }

In the example above, an asterick (*) is used as universal selector

Descendant Selectors

If we need to style an element nested inside another element.

Example: If we want to style li elements inside ul element, ul li { width: 100px } can be used.

Class Selectors

Based on the values of class attribute, the styles can be applied selecting that class attribute value.

Example: Let's say we have a tag, <p class="para">, we can select the para class using CSS and style the p tag: .para { color: red; }

ID Selectors

Like class selectors, id attribute's value can also be used as selector and style the element with id attribute.

Example: In an HTML element, <h1 id="heading1">, we can use CSS to select heading1 and style h1: #heading1 { color: purple; }

Child Selectors

Child Selectors act similar to descendant selectors except child selectors select the direct child of the selector and not the ones nested inside other elements.

Example: Let's say we have following HTML elements: 

          <div class="paras">
            <p> This is a para </p>
            <p> This is para2 </p>
            <div class="next_para">
              <p> This is inside next_para </p>
            </div>
          </div>

If we use child selector: .paras > p { color: blue; }, then it would not affect the p tag inside div with next_para class as it is not the direct child of paras div.

However, if we use descendant selector, .paras p { color: blue; }, then it would apply the style to all the p tags inside paras class even if it is not the direct child.

Attribute Selector

Sometimes, there may be need to apply styles to particular attributes.

Example: If we want to select all inputs with type text, we can use: input[type="text"] { width: 200px; }

In the example above, input[type="text"] is an attribute selector.

Commenting CSS

/*.....*/ can be used to comment CSS styles.

Example: table { /* border-color: red; */ }

CSS Inclusion

CSS can be used in an HTML document in following ways:

Inline CSS

CSS can be applied to an HTML element in the element itself using style attribute.

Example: <p style="color: red;">This is a red text</p>

Internal CSS

Also known as embedded CSS, the styles are applied to all the elements inside a particular document.

Example: Let's assume we have an HTML document index.html and we are applying embedded CSS to the page using:

          <head>
            <style type="text/css" media="all">
              h1 {
                color: white;
                background-color: blue;
              }
            </style>
          </head>

In the example above, the style will be applied to all the h1 tags in index.html. However, if we have h1 tag in another HTML document contact.html, then we need to style h1 in contact.html again.

External CSS

Using Embedded CSS would be okay if we don't want to have styles limited to a particular HTML document.

However, if we have to use the same styles again and again in other documents as well then, the code replication is not a good practice.

To resolve this, we can create an external CSS and link it to any HTML document that requires the styles using <link> tag.

Example:<link type="text/css" href="styles.css" media="all" />

The example above links an external CSS file styles.css. <link> tag should be used inside head element.

Imported CSS

@import can be used to import an external CSS file/stylesheet similar to <link> element.

Example: @import "styles.css";

Overriding CSS Rules

As we can see different methods can be used to include styles in an HTML document. Let's see the overriding rules:

  • Any inline style takes the highest priority i.e., if an element has inline style and also has style declared in embedded style element or in an external stylesheet, then the inline style will override other styles.
  • Any rule defined in embedded style element will override rules defined in an external stylesheet.
  • Any rule defined in an external CSS takes the least priority and will be applied only when the above two rules are not applicable.

CSS Measurement Units

Measurement Units play a very important role in creating an attractive and responsive design.

  Unit Description Example
Absolute Units
cm Defines a measurement in centimeters (cm) table { margin-top: 20cm }
mm Defines a measurement in millimeters (mm) table { margin-top: 20mm }
in Defines a measurement in inches (in) table { margin-top: 2in }
px Defines a measurement in pixels (px=1/96 of 1in) table { margin-top: px }
pt Defines a measurement in points (pt=1/72 of 1in) table { margin-top: 20pt }
Relative Units
% Defines a measurement as percentage relative to another value h1 { font-size: 20px; line-height: 50% }
em Defines a measurement relative to the font-size of the element p { font-size:10px; line-height: 2em }
here, 2em = 2x10=20px
rem Defines a measurement relative to the font-size of the root element html { font-size: 16px; } table td { font-size: 3rem; }
here, since html is the root element, td of table will have 3rem = 3x16=48px
vw Defines a measurement relative to the viewport width of the device h1 { font-size:10vw; }
vh Defines a measurement relative to the viewport height of the device h1 { font-size:10vh; }

CSS Colors

CSS uses color values to specify a color for text color, border color or for the background of any element.

CSS uses different formats to specify color values.

Format Syntax Example
Keyword Name of color (black, white, red, etc.) p { color: white; }
Hex Code #RRGGBB p { color: #FFFFFF; }
Short Hex Code #RGB p { color: #FFF; }
RGB rgb(rrr, ggg, bbb) p { color: rgb(255,255,255); }
RGBA rgba(rrr, ggg, bbb, a); a stands for alpha p { rgba(255, 255, 255, 1); }

CSS Backgrounds

CSS allows user to set background for any element in HTML using different background properties.

Background Properties

  • background-color: Used to set the background color of an element.
  • background-image: Used to set the background image of an element.
  • background-repeat: Used to control the repetition of the background image. Values: repeat, repeat-x, repeat-y, no-repeat
  • background-position: Used to control the position of the background image. Values: left top, left center, left bottom, right top, right center, right bottom, center top, center center, center bottom
  • background-attachment: Used to control the scrolling of the background image. Values are fixed, scroll
  • background-size: Used to control the scrolling of the background image. Values are cover, contain

Examples:

Set the Background Color: 

          <p style="background-color: lightblue">
            This text has a lightblue background. 
          </p>

Output:

This text has a lightblue background.

Set the Background Image: 

          <div style="color: #FFF; background-image: url('../Day3/tech.jpg')">
            This div has a background image. 
          </div>

Output:

This div has a background image.

Set the Background Repeat: 

          <div style="height: 200px; color: #FFF; background-image: url('../Day3/tech.jpg'); background-repeat: repeat;">
            This div has a background image. 
          </div>

Output:

This div has a background repeat.

Set the Background Position: 

          <div style="height: 200px; color: #FFF; background-image: url('../Day3/tech.jpg'); background-position: top center;">
            This div has a background position set to center center. 
          </div>

Output:

This div has a background position set to center center.

Set the Background Attachment: 

          <div style="height: 200px; color: #FFF; background-image: url('../Day3/tech.jpg'); background-position: top center; background-attachment: fixed">
            <p>This div has a background attachment set to fixed. </p>
          </div>

Output:

This div has a background attachment set to fixed.

Set the Background Size: 

          <div style="height: 200px; color: #FFF; background-image: url('../Day3/tech.jpg'); background-position: center; background-repeat: no-repeat; background-size: cover;">
            <p>This div has a background size set to cover. </p>
          </div>

Output:

This div has a background size set to cover.

Set the Background using shorthand property:

When using shorthand property, the order of the property values must be:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position 

          <div style="height: 200px; color: #FFF; background: url('../Day3/tech.jpg') no-repeat scroll center;">
            <p>This div has a background image set using shorthand property. </p>
          </div>

CSS Text

CSS can be used to manipulate text using different text properties.

  • color: Used to set the color of the text.
  • text-align: Used to set the align of the text
  • text-indent: Used to indent the text in a paragraph.
  • text-decoration: Used to underline, overline and strikethrough text.
  • text-transform: Used to capitalize text, convert text to lowercase or uppercase letters.
  • letter-spacing: Used to manage spaces between letters in a word.
  • word-spacing: Used to manage spaces between words in a paragraph.
  • text-shadow: Used to set the text shadow around the text.

Examples:

Set the text color: 

          <p style="color: red;"> This is a red text. </p>

Output:

This is a red text.

Set the text alignment: 

          <p style="text-align: center;"> This is a centered text. </p>

Output:

This is a centered text.

Set the text indent: 

          <p style="text-indent: 15px;"> This is an indented text <br> and this is text under that. </p>

Output:

This is an indented text
and this is text under that.

Set the text decoration: 

          <p style="text-decoration: underline;"> This is an underlined text. </p>

Output:

This is an underlined text.

Set the text transform: 

          <p style="text-tranform: uppercase;"> This text will be in uppercase. </p>

Output:

This text will be in uppercase.

Set the letter spacing: 

          <p style="letter-spacing: 5px;"> This text is using letter spacing. </p>

Output:

This text is using letter spacing.

Set the word spacing: 

          <p style="word-spacing: 5px;"> This text is using word spacing. </p>

Output:

This text is using word spacing.

Set the text shadow: 

          <p style="text-shadow: 2px 5px 3px blue;"> This text is using text shadow. </p>

Output:

This text is using text shadow.

In the example above, 2px is the horizontal offset, 5px is the vertical offset, 3px is the blur and blue is the color.

CSS Fonts

CSS can also be used to set different font styles to a text or paragraph in HTML.

  • font-size: Used to manage the size of a font
  • font-weight: Used to manage the boldness of a font
  • font-family: Used to set the face of a font
  • font-style: Used to make a font italic or oblique
  • font-variant: Used to create a small-caps effect

Examples:

Set the font size

<p style="font-size: 20px;"> This text has font size 20px. </p>

Output:

This text has font size 20px.

Set the font weight

<p style="font-weight: 800;"> This text has font weight 800. </p>

Output:

This text has font weight 800.

Set the font family

<p style="font-family: Verdana;"> This text has font family Verdana. </p>

Output:

This text has font family Verdana.

Set the font style

<p style="font-style: italic;"> This text has font style italic. </p>

Output:

This text has font style italic.

Set the font variant

<p style="font-variant: small-caps;"> This text has font variant small-caps. </p>

Output:

This text has font variant small-caps.
