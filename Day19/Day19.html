JavaScript Arrays

Arrays are complex variables that allow us to store more that one value or group of values in a single variable name.

Syntax:

var myArray = [element0, element1, ....., elementN]; OR var myArray = new Array(element0, element1, ....., elementN);

Example:

var fruits = ["Mango", "Banana", "Grapes"]; OR var fruits = new Array("Mango", "Banana", "Grapes");

Accessing the Elements of an Array

Array elements can be accessed by their index. Array indexes start from 0 and not 1.

Example: document.write(fruits[0]); document.write(fruits[2]);

Getting the length of an array

The length property returns the length of an array.

Example: document.write(fruits.length);

Adding New Elements to an Array

To add a new element to an existing array, we use push() method.

Example: fruits.push("Orange");

Adding New Element to beginning of an Array

To add a new element to the beginning of an existing array, we use unshift() method.

Example: fruits.unshift("Orange");

Removing Elements from an Array

To remove the last element from an existing array, we use pop() method.

Example: fruits.pop();

Removing first element from an Array

To remove the first element from an array, we use shift() method.

Example: fruits.shift();

Creating a String from an Array

The join() method is used to create a string by joining the elements of an array.

Syntax: myArray.join()

Example: fruits.join()

Extracting a Portion of an Array

The slice() method is used to extract out a portion of an array but keep the original array intact.

Syntax: myArray.slice(startIndex, endIndex)

Example: fruits.slice(1,3)

Merging Two or More Arrays

The concat() method can be used to merge or combine two or more arrays.

Syntax: var newArray = myArray1.concat(myArray2);

Searching through an Array

The indexOf() and lastIndexOf() methods can be used to search an array for a specific value.

Syntax: myArray.indexOf(element);

includes() method is also used to find out whether an array includes a certain element or not.

Syntax: myArray.includes(element);

Sorting Arrays

Sorting means managing array elements in alphabetical order.

Sorting String Array

The sort() method is used for sorting array elements in alphabetical order.

Syntax: myArray.sort()

Reverse Sorting String Array

The reverse() method is used for sorting array elements in reverse order.

Syntax: myArray.reverse()

Sorting Numeric Array

The sort() method is used for sorting array elements in alphabetical order. But since 1 comes before 2 so when using sort() method, 100 can come before 20.

To fix this, we pass a compare function as below:

Syntax: myArray.sort(function(a,b){ return a-b; })

Finding the Maximum and Minimum Value in an array

The apply() method can be used in combination with Math.max() and Math.min() to find the maximum and minimum value inside an array.

Syntax: Math.max.apply(null, array); Math.min.apply(null, array);

JavaScript Objects

JavaScript object is a collection of named values. It is similar to arrays, the difference being the keys can be defined ourselves.

Creating Objects:

An object can be created with curly brackets ({}).

Syntax: var myObj = { key1:value1, key2:value2, ....., keyN:valueN };

Example:

var person = { "first_name": "John", "last_name": "Doe", "gender": "Male", "age": 30 };

Accessing Object's Properties

To access or get the value of a property, the dot(.) or square bracket notation ([]) can be used.

Example: person.first_name; person["age"];

Adding or Updating Object's Properties

We can set the new properties or update the existing one using the dot (.) or bracket ([]) notation.

Example: person.height="5.5"; person.age = 40; person["gender"] = "Female";

Deleting Object's Properties

The delete operator can be used to completely remove properties from an object.

Example: delete person.age;

JavaScript Loops

Loops are used to execute the same block of code again and again until a certain condition is met.

Types of Loops in JavaScript

  • while
  • do...while
  • for
  • for...in

The while Loop

The while loop loops through a block of code until the given condition is true.

Syntax:

while(condition) { // block of code to be executed }

Example:

var i = 1; while(i <= 5) { document.write("This is a while loop"); i++; }

The do...while Loop

The do...while loop is a variant of while loop; only difference is do...while loop executes the block of code once before evaluating the condition.

Syntax:

do { // block of code to be executed } while(condition);

Example:

var i = 1; do { document.write("This is a while loop"); i++; } while(i <= 5);

The for Loop

The for loop repeats a block of code until the given condition is true for a certain number of times.

Syntax:

for(initialization; condition; increment) { // block of code to be executed }

Example:

for(var i=1; i<=5; i++) { document.write("This is a while loop"); }

The for..in Loop

The for..in loop is a special type of loop that iterates over the properties of an object, or elements of an array.

Syntax:

for(variable in object) { // block of code to be executed }

Example:

var fruits = ["Orange", "Banana", "Grapes"] for(var i in fruits) { document.write(fruit[i]); } var person = { "first_name": "John", "last_name": "Doe", "gender": "Male", "age": 30 } for(var key in person) { document.write(key + ":" + person[key]); }

JavaScript Functions

A function is a group of statements that perform specific tasks and can be kept and maintained separately from main program.

Advantages of using functions:

  • Reduces the repetition of code within a program
  • Makes the code easier to maintain
  • Makes it easier to handle errors

Defining and Calling a function

The declaration of a function start with the function keyword, followed by the name of the function, followed by parentheses.

Syntax:

function functionName() { // Code to be executed }

Example:

function helloWorld() { alert("Hello, World!"); }

Parameters in Functions

We can specify any number of parameters when defining a function to accept input values at run time.

Syntax:

function functionName(parameter1, parameter2, parameter3..., parameterN) { // Code to be executed }

Example:

function displaySum(num1, num2) { var total = num1 + num2; alert(total); } displaySum(10, 15);

JavaScript Events

An event is something that occurs when user interact with the web page i.e., when a button is clicked, text is entered in a input field, key is pressed, etc.

Mouse Events

A mouse event is triggered whn the user click some element, hover over some elements, etc.

The Click Event (onclick)

The click event occurs when a user clicks on an element on an web page.

Example: <button onclick="alert('Button Clicked')"> Click Here </button>

The Contextmenu Event (oncontextmenu)

The contextmenu event occurs when a user right clicks on an element to open a context menu.

Example: <button oncontextmenu="alert('Button Clicked')"> Click Here </button>

The Mouseover Event (onmouseover)

The mouseover event occurs when a user moves the mouse pointer over an element.

Example: <button onmouseover="alert('Mouseover')"> Click Here </button>

The Mouseout Event (onmouseout)

The mouseout event occurs when a user moves the mouse pointer leaves an element.

Example: <button onmouseout="alert('Mouseout')"> Click Here </button>

Keyboard Events

A keyboard event is fired when the user press or release a key on the keyboard

The keydown Event (onkeydown)

The keydown event occurs when the user presses down a key on the keyboard.

Example: <input type="text" onkeydown="alert('key down')">

The keyup Event (onkeydown)

The keyup event occurs when the user releases a key on the keyboard.

Example: <input type="text" onkeyup="alert('key down')">

The keypress Event (onkeypress)

The keypress event occurs when the user presses down a key on the keyboard that has a character value associated with it. Keys like Esc, Ctrl, Shift, Alt, Arrow keys, etc. don't fire keypress event.

Example: <input type="text" onkeypress="alert('key pressed')">

Form Events

A form event is fired when a form control receives or loses focus or when the user modify the text input, submit a form, etc.

The focus Event (onfocus)

The focus event occurs when the user gives focus to an element in a form.

Example: <input type="text" onfocus="alert('focused')">

The blur Event (onblur)

The blur event occurs when the user loses focus to an element in a form.

Example: <input type="text" onblur="alert('blurred')">

The change Event (onchange)

The change event occurs when the user changes the value of an element in a form.

Example: <input type="text" onchange="alert('changed')">

The submit Event (onsubmit)

The submit event occurs when the user submits a form.

Example: <form onsubmit="alert('submitted')"></form>
