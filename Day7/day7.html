CSS Margins

CSS margin properties are used to create space around elements, outside the element.

CSS margin Properties:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
  • margin(shorthand)

All the margin properties can have auto, length in px, pt, cm, etc., % and inherit values. Also, negative values are allowed too.

Example:

<div style="background-color: blue; color: white; margin-top:20px; margin-right: 30px; margin-bottom: 25px; margin-left: 40px;"> This section has margins implemented. </div>

CSS Borders

CSS border properties are used to specify the style, size and color of an element's border.

CSS border Properties:

  • border-style
  • border-width
  • border-color
  • border(shorthand)
  • border-radius

CSS border-style Property:

The border-style property accepts following values:

  • dotted
  • dashed
  • solid
  • double
  • groove
  • ridge
  • inset
  • outset
  • none
  • hidden

CSS border Property:

The border property is the shorthand property to define a border around an element in CSS. The border property should be written in the following order: border-width, border-style and border-color.

Example of border property:

<div style="border: 5px solid blue"> This section has blue border. </div>

Output:

This section has blue border.

Borders on Individual sides:

If there is a need to have different border styles, widths or colors for different sides, we can use following properties:

  • border-top-style/width/color
  • border-right-style/width/color
  • border-bottom-style/width/color
  • border-left-style/width/color

Example:

<div style="border-top-style: dotted; border-right-style: double; border-bottom-style: solid; border-left-style: groove; border-top-color: blue; border-right-color: red; border-bottom-color: yellow; border-left-color: purple;"> This section has different borders on different sides. </div>

Output:

This section has different borders on different sides.

border-radius Property:

The border-radius property is used to add rounded borders to an element.

Examples

<div style="border: 3px solid red; padding: 10px;"> This is normal border. </div>

This is normal border.

<div style="border: 3px solid red; padding: 10px; border-radius: 10px;"> This is rounded border. </div>

This is rounded border.

CSS Dimensions

CSS dimension properties are used to control the dimensions of an element.

CSS Dimension Properties:

  • height: Set the height of an element
  • width: Set the width of an element
  • line-height: Set the height of a line in a text
  • max-height: Set the maximum height of an element.
  • min-height: Set the minimum height of an element.
  • max-width: Set the maximum width of an element.
  • min-width: Set the minimum width of an element.

height and width properties:

height and width properties sets the height and width of a box and can take values of a length, percentage and auto.

Example:

<div style="width: 20%; height: 300px; border: 1px solid red;"> </div>

Output:

line-height property:

line-height property sets the height of the line in a text and can take values of a length or a percentage.

Example:

<div style="width: 20%; height: 100px; border: 1px solid red;"> <p style="line-height: 30px"> Here is the first line of text <br> Here is the second line of text. </p> </div>

Output:

Here is the second line of text.

CSS Shadow

CSS shadow properties are used to apply shadows to an element.

CSS shadow Properties:

  • text-shadow
  • box-shadow

box-shadow property

The box-shadow property is used to apply shadows to an element.

Example:

<div style="width: 20%; height: 100px; color:#fff; background-color: #002547; box-shadow: 3px 5px 10px red"> This box is using box-shadow. </div>

Output:

This box is using box-shadow.

CSS Lists

CSS list properties are used to control list type, position, style of a list.

CSS Lists Properties:

  • list-style-type
  • list-style-position
  • list-style-image
  • list-style (shorthand)

list-style-type property

The list-style-type property is used to control the style of bullet points in unordered lists or the style of numbering characters in ordered lists.

Unordered list-style-type values

  • none
  • disc
  • circle
  • square

Ordered list-style-type values

  • decimal
  • decimal-leading-zero
  • lower-alpha
  • upper-alpha
  • lower-roman
  • upper-roman

list-style-position property

The list-style-position property is used to control the position of the marker or the numbering characters.

Unordered list-style-type values

  • none
  • inside
  • outside

Example:

<ul style="list-style-type: circle; list-style-position: outside;"> <li> First Outside Element </li> <li> Second Outside Element </li> </ul> <ol style="list-style-type: upper-alpha; list-style-position: inside;"> <li> First Inside Element </li> <li> Second Inside Element </li> </ol>

Output:

  • First Outside Element
  • Second Outside Element
  1. First Inside Element
  2. Second Inside Element

list-style-image property

The list-style-image property is used to specify an image for the marker or the numbering characters.

Example:

<ul style="list-style-image: url('list_style_image.gif');"> <li> First Element </li> <li> Second Element </li> </ul>

Output:

  • First Element
  • Second Element

list-style shorthand property

The list-style property is used to set one or more list properties in a single property.

Example:

<ul style="list-style: inside square;"> <li> First Element </li> <li> Second Element </li> </ul>

Output:

  • First Element
  • Second Element
