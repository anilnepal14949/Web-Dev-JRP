PHP Date/Time

PHP date() function is used to convert timestamp to a more readable date and time. The data is stored in a format called UNIX Timestamp.

Syntax:

        date(format, timestamp)

Formatting Date and Time with PHP

The format parameter of the date() function is a string containing multiple characters used to generate a human readable date string.

Date-Related Formatting Characters

  • d - Represent day of the month; two digits with leading zeros
  • D - Represent day in the week (three characters)
  • l - Represent full day in the week
  • m - Represent month in numbers with leading zeros
  • j - Represent dat of month without leading zeros
  • S - Represent English Ordinal Suffix (st, nd, rd or th)
  • M - Represent month in text, abbreviated
  • F - Represent full month in text
  • y - Represent year in two digits
  • Y - Represent year in four digits

Time-Related Formatting Characters

  • h - Represent hour in 12-hour format with leading zeros
  • H - Represent hour in 25-hour format with leading zeros
  • i - Represent minutes with leading zeros
  • s - Represent seconds with leading zeros
  • a - Represent lowercase am or pm
  • A - Represent uppercase am or pm

time() Function

The time() function is used to get the current time as a UNIX timestamp.

We can use date() function to get the formatted date and time.

mktime() Function

We can also use mktime() function to create the timestamp based on a specific date and time.

Syntax: mktime(hour, minute, second, month, day, year)

Example:mktime(15, 20, 12, 10, 24, 2022)

PHP Include Files

PHP allows to include the code from another PHP file within another PHP file.

The include() and require() are used to include the code from another file to one file in PHP.

Difference between include() and require()

If the included file is not found, the include() statement will only generate a warning but allows to execute the script, however the require() statement generates a fatal error and stops the execution.

include_once() and require_once() statements

Sometimes, if the application is very huge, same file might have been included in multiple places and there may arise conflicts.

To prevent such situation, PHP provides include_once() and require_once() statements which ignore the include statements if it is already included.

PHP Files

PHP has support for working with files and directories stored on the web server.

Opening a File

Before working on any file, first the file has to be opened. The PHP fopen() function is used to open a file.

Syntax: fopen(filename, mode)

File Modes:

  • r: Open the file for reading only
  • r+: Open the file for reading and writing
  • w: Open the file for writing only and clears the contents of the file.
  • w+: Open the file for reading and writing and clears the contents of the file.
  • a: Opens the file for writing only. Writes the contents to the end of file.
  • a+: Opens the file for reading and writing. Writes the contents to the end of file.
    • NB: If the file is not found, PHP will try to create it in w, w+, a, a+ modes.

file_exists() Function

PHP generates a warning message if we try to open the file that doesn't exist.

To avoid these error messages, PHP has file_exists() function to check if the file exists or not.

Closing a File

Once we are done with working with a file, it needs to be closed.

The fclose() function is used to close a file.

Reading from Files

The fread() function can be used to read contents from a file.

Syntax: fread(file handle, length in bytes)

filesize() Function

If we need to read entire contents from a file, we can use fread() function in conjunction with filesize() function.

Example: fread($handle, filesize($file))

readfile() Function

However, we can also use readfile() function to read the entire contents from a file and display it. It is one of the easiest ways to read the contents from a file.

Example: readfile($file)

file_get_contents() Function

However, we can also use file_get_contents() function to read the entire contents from a file and store in a variable as a single string.

Example: file_get_contents($file)

file() Function

However, PHP also provides file() function to read the entire contents from a file and store in a variable as array of lines.

Example: file($file)

Writing to a file

The PHP fwrite() function can be used to write or append contents in a file.

Syntax: fwrite(file handle, string)

file_put_contents() Function

The file_put_contents() can also be used to write or append contents into a file.

Syntax: file_put_contents($file, $data)

This function overwrites the contents in the file. To append the contents, we need to pass the special FILE_APPEND flag as third parameter.

Syntax: file_put_contents($file, $data, FILE_APPEND)

Remove Files

PHP provides unlink() function to delete files or directories.

Syntax: unlink($file)

PHP Cookies

A cookie is a small text file that lets you store a small amount of data on the user's computer.

Setting a Cookie

The setcookie() function is used to set a cookie in PHP.

Syntax: setcookie(name, value, expire, path, domain, secure)

Parameters

  • name: The name of the cookie
  • value: The value of the cookie
  • expires: The expiry date in UNIX timestamp format
  • path: Specify the path on the server for which the cookie will be available.
  • domain: Specify the domain for which the cookie will be available to.
  • secure: Enable this if the cookies are to be sent only for a HTTPS connection.

Accessing Cookies Values

The PHP $_COOKIE superglobal variable is used to retrieve a cookie value.

Example: print_r($_COOKIE);

The isset() Function

It's a good practice to check whether a cookie is set or not. To do this, we can use isset() function.

Example: isset($_COOKIE["name"])

Removing Cookies

We can delete a cookie by calling the setcookie() function with the cookie name, an empty value and expiry set in the past.

Example: setcookie("name", "", time()-3600)

PHP Sessions

Although, cookies can be used for storing data, it has some security issues. Since cookies are stored on the user's computer, it is possible for an attacker to easily modify a cookie content to insert harmful data in the application that might break the application.

This can be solved by using the PHP session. PHP session stores data on the server rather that user's computer.

For each session, a unique session ID would be generated which is almost impossible to guess for attackers.

Starting a PHP Session

Before we can store any information in session variables, we must first start a session. To begin a new session, we can call the PHP session_start() function.

NB: The session_start() function must be called at the beginning of the page or before any output is generated in the browser.

Storing and Accessing Session Data

We can store all session data as key-value pairs in the $_SESSION[] superglobal array.

Example: $_SESSION["name"] = "Your Name";

In order to access the session stored data, we can again use the $_SESSION[] superglobal array and pass the key that needs to be accessed.

Example: echo $_SESSION["name"];

Destroying a Session

We can use unset() function to remove certain session data.

Example: unset($_SESSION["name"])

However, to destroy a session completely, we can call the session_destroy() function.
