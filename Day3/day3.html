HTML Links

HTML Links

Anchor tag, <a>...</a> is used to create links in HTML. Links or hyperlinks can used to navigate between different webpages or within same webpage by clicking on the text or images on the webpage.

Examples: <a href="https://google.com"> Go to Google </a> displays the following link that can redirect our webpage to google.com.

Go to Google

Attributes

  • href
  • target

Links within a webpage

Hyperlinks can also be used to link to various sections in a single webpage. Example: Go to bottom link below is linked to a section in this webpage. When clicked, the webpage scrolls to that linked section.

Go to bottom

Image Links

Not only text, images can also be used to link to different webpages or various sections in a single webpage. Example: Image below is linked to open google homepage when clicked.

Email Links

Anchor tag also provides an option to specify an email address and send an email to the specified email. In order to send email, use mailto: email address as value in href attribute.

Example: Send an email to abc@gmail.com

The code above opens a default email client on the system like outlook.

Download Links

We can also use links to download files by simply clicking on it. To download a file, the value for the href attribute must be set to the full path of the file.

Example: Download PDF

HTML iFrames

HTML iFrames

iFrames or inline frames are used to embed another document/webpage within the current document/webpage.

Attributes

  • src
  • name
  • width
  • height

Example: <iframe src="iframe.html"></iframe>

HTML Embed

HTML Embed

<embed> tag can be used to include audio or video into a webpage. In case a browser doesn't support <embed> tag, <noembed> tag can be used to display a message or an image stating no support for embed tag.

Attributes

  • src
  • name
  • width
  • height
  • loop
  • volume

Example: <embed src="sample_video.mov"></embed> embeds sample_video.mov in the webpage. Since, HTML5 has introduced <video> and <audio> tags for including videos and audios in the webpage, it is better recommended to use these tags instead of embed tag.

HTML Marquees

HTML Marquees

HTML Marquees are used to display scrolling text either horizontally across or vertically down the webpage.

Example: <marquee> This is a scrolling text </marquee>

Attributes

  • width
  • height
  • direction
  • behavior
  • scrolldelay
  • scrollamount
  • loop
  • bgcolor
  • hspace
  • vspace

Example using attributes:

<marquee width="500" height="30" direction="left" behavior="alternate" scrolldelay="10" scrollamount="10" loop="0" bgcolor="yellow" hspace="5" vspace="5"> This is an example of marquee tag </marquee>

This is an example of marquee tag

HTML Blocks

HTML Blocks

<div> and <span> tags can be used to create block-level and inline element respectively and apply styles to the blocks individually.

Example of div tag: <div> This is a block </div>

Example of span tag: <p> This is a <span> span. </span></p>

NB: These tags will make more relevance after we start styling these elements using CSS
