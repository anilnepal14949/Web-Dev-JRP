PHP Introduction

PHP stands for HyperText Preprocessor. It is a very popular and widely-used open source server-side scripting language used to write dynamically generated web pages.

What We Can Do with PHP?

  • Generate pages and files dynamically
  • Create, open, ready, write and close files
  • Send emails to the users
  • Store, delete and modify information in the database
  • Restrict unauthorized access to your website

PHP Syntax

A PHP script starts with the tag; known as delimiter () tells the PHP engine to treat the enclosed code block as PHP code, rather than simple HTML.

Every PHP statement must end with a semicolon (;).

PHP Comments

A comment is simple the text that is ignored by the PHP engine.

PHP supports both single line comment (//, #) or a multi-line comments (/*..*/).

Case Sensitivity

Like JavaScript, PHP is also a case-sensitive language; meaning that variables $color, $Color, $COLOR are all different. The same applies for methods too.

PHP Variables

Variables are used to store data like string of text, numbers, etc. The values of variables can change during the script execution.

Things to know about PHP Variables

  • No need to declare before adding a value to it; automatically converts the variable to the correct data type, depending on its value.
  • Once declared, it can be used throughout the script
  • The assignment operator (=) is used to assign value to a variable.

Naming Conventions

  • All variables in PHP start with a $ sign
  • A variable name must start with a letter or an underscore (_) character.
  • A variable name cannot start with a number
  • A variable name in PHP can only contain alpha-numeric characters and underscores (A-z, 0-9 and _)
  • A variable name cannot contain spaces.
NB: Variable names are case-sensitive.

PHP Constants

A constant is a name or an identifier for a fixed value. Constants are like variables, except that the values cannot be changed after declaration.

PHP Constants are defined using PHP's define() function, which accepts two arguments: the name of the constant and its value.

Example: define("DB_USER", "admin"); echo DB_USER;

NB: There is no need of $ symbol when using the constant.

PHP Echo and Print Statements

The echo statement can output one or more strings.

Echo is a language construct not a function so can be used without parentheses.

PHP print Statement

The print statement is an alternative to echo statement to display output to the browser.

Difference between echo and print: print returns 1 after displaying the output while echo doesn't return anything. Hence, echo is faster than print and used commonly compared to print statement.

PHP Data Types

Different types of data can be stored in PHP variables like string, numbers, array, objects, etc.

Data types supported by PHP: integer, floating point, string, booleans, array, object, resource and null.

PHP Integers

Integers are whole numbers, without a decimal point.

Example: $a = 123; $b = -50;

PHP Strings

Strings are sequences of characters (letters, numbers and special characters). The strings are enclosed in double quotes ("") or single quotes('').

Example: $a = "Hello"; $b = 'World';

PHP Floating Point Numbers

FLoating point numbers (floats, doubles or real numbers) are decimal or fractional numbers.

Example: $a = 1.23; $b = 5.658685;

PHP Booleans

Booleans are like a switch with two possible values either 1 (true) or 0 (false).

Example: $a = true; $b = false;

PHP Arrays

An array is a variable that can hold more than one value at a time.

Example: $a = array("hello", "world");

PHP Objects

An object is a specific instance of a class. Every object has properties and methods corresponding to those of its parent class.

Example: $car = new Car;

PHP NULL

NULL is a special data type used to represent empty variables in PHP.

Example: $a = NULL;

PHP Resources

Resources are special variables, holding a reference to an external resource. Eg., when opening a file or connecting to a database.

Example: $conn = mysqli_connect("localhost", "root", "");

PHP Conditionals

Conditional statements are used to perform different ations based on the results of a logical or comparative test conditions at run time.

The if Statement

This is the simplest conditional statement used to run a block of code only if the specified condition evaluates to true.

Syntax:

 
        if(condition) { 
          //code to be executed 
        }

The if...else Statement

The if...else statement allows to run a block of code if the specified condition evaluates to true and another block of code if it evaluates to false.

Syntax:

 
        if(condition) { 
          //code to be executed if condition is true 
        } else {
          //code to be executed if condition is false
        }

The if...elseif...else Statement

This statement used to combine multiple if...else statements.

Syntax:

 
        if(condition1) { 
          //code to be executed condition1 is true
        } else if(condition2) {
          //code to be executed condition2 is true
        } else {
          //code to be executed if both condition1 and condition2 are false
        }

The Ternary Operator

The ternary operator is a shorthand way of writing if...else statements.

Syntax:

(condition)?true:false

The Switch...Case Statement

This statement is similar to if...elseif...else statement.

Syntax:

        switch(n) {
          case label1:
            // code to be executed if n=label1
            break;
          case label2:
            // code to be executed if n=label2
            break;
          ...
          default:
            // code to be executed if n doesn't match any label
        }

PHP Arrays

Arrays are complex variables that allow us to store more than one value or a group of values under a single variable name.

Types of Arrays in PHP

  • Indexed Array: An array with a numeric key.
  • Associative Array: An array where each key has its own specific value
  • Multidimensional array: An array containing one or more arrays within itself.

Viewing Array Structure and Values:

We can use statements var_dump(array_name) or print_r(array_name) to display the structure and values of any array.

Sorting Arrays

PHP comes with a number of built-in functions to sort any array elements in different ways like alphabetically or numerically in ascending or descending order.

  • sort() and rsort()
  • asort() and arsort()
  • ksort() and krsort()

PHP Loops

Loops are used to execute the same block of code repeatedly, until a certain condition is met.

Types of Loops in PHP

  • while
  • do...while
  • for
  • foreach

while Loop

The while statement loops through a block of statements as long as the condition specified in the while loop evaluates to true.

Syntax:

        while(condition) {
          // code to be executed
        }

do...while Loop

The do...while statement is a variant of while loop; loops through a block of statements as long as the condition specified in the while loop at the end of each loop evaluates to true.

NB: A do...while loop executes the code once, then evaluates the condition.

Syntax:

        do {
          // code to be executed
        } while(condition)

for Loop

The for statement loops through a block of statements for a certain number of times until a condition is met.

Syntax:

        for(initialization; condition; increment) {
          // code to be executed
        }

foreach Loop

The foreach statement is used to loop through arrays.

Syntax:

        foreach($array as $value) {
          // code to be executed
        }

Also another syntax used for associative arrays if there is need to access keys and values both.

        foreach($array as $key => $value) {
          // code to be executed
        }

PHP Functions

A function is a self-contained block of code that performs a specific task.

PHP has a huge collection of in-built functions that can be called directly without declaring it like print_r(), var_dump(), etc.

PHP User-Defined Functions

PHP allows to define our own functions apart from the built-in functions.

Creating and Invoking Functions

Syntax: 

        // Creating function
        function functionName() {
          // code to be executed
        }
        // Invoking/calling function
        functionName();

Functions with Parameters

We can specify parameters when we define a funciton to accept input values at run time.

Syntax:

        // creating function with parameters
        function myFunc($para1, $para2 ... , $paraN) {
          // code to be executed
        }

        // invoking function
        myFunc($arg1, $arg2, ... , $argN);

Returning Values from a Function

A function can return a value back to the script that called the function using the return statement.

Syntax:

        function myFunc($para1, $para2) {
          // code to be executed
          return $result; // could be numbers, strings, arrays or objects
        }

Understanding the Variable Scope

We can declare as many variables as we want in a PHP Script. However, the location of the declaration determines the accessibility of the variable known as variable scope.

Variables defined inside a function block is not accessible from outside the block.

Example:

        // Defining function to calculate total
        function total($num1, $num2) {
          $result = $num1 + $num2;
          echo $result;
        }

        total(10, 20); // Outputs: 30
        echo $result; // generates undefined variable error

The global Keyword

Sometimes, there may be need to access the variables from the main block into a function or vice-versa. PHP provides the global keyword in such situations.

Example: 

        // Defining function to calculate total
        function total($num1, $num2) {
          global $result;
          $result = $num1 + $num2;
          echo $result;
        }

        total(10, 20); // Outputs: 30
        echo $result; // Outputs: 30

Recursive Functions

A recursive function is a function that calls itself again and again until a certain condition is satisfied.

Recursive functions are often used to solve complex mathematical calculations like factorial, or to process deeply nested structures like printing all values of a deeply nested array.
