PHP Static Methods

Static methods are the type of methods that can be called without creating an instance or object of the class.

The static keyword is used for declaring any method as static.

Syntax:

        class StaticExample {
          public static function staticMethod() {
            // code goes here
          }
        }

Accessing static methods

Example:

        StaticExample::staticMethod();

Accessing static methods from outside the class

Example:

        StaticExample::staticMethod();

Accessing static methods from another class

Example:

        class NewClass {
          public function __construct() {
            StaticExample::staticMethod();
          }
        }

Accessing static methods from child class

Example:

        class ChildClass extends StaticExample {
          public function __construct() {
            parent::staticMethod();
          }
        }

PHP Static Properties

Static properties are similar to static methods that can be called without creating an instance or object of the class.

The static keyword is used for declaring any property as static.

Syntax:

        class StaticExample {
          public static $staticProperty = "Hello From Static";
          public static function staticMethod() {
            // code goes here
          }
        }

Accessing static property from outside the class

Example:

        StaticExample::$staticProperty;

Accessing static properties from methods in the same class

Example:

        self::$staticProperty;

Accessing static properties from child class

Example:

        class ChildClass extends StaticExample {
          public function __construct() {
            return parent::$staticProperty;
          }
        }

PHP Namespaces

Namespaces are qualifiers used for group classes that work together and allow same name to be used for more than one class.

The namespace keyword is used for declaring namespaces.

Declaring Namespace

Example:

        namespace Examples;
        class First {
          // code goes here
        }

NB: Namespace declaration must be the first thing in the PHP file.

One namespace is used, the classes and functions belong to that namespace.

Accessing classes or methods from same class doesn't change, however accessing classes and methods from outside the class should be done using following:

Accessing classes outside the namespace

Example:

        $example = new Examples\First();

Namespace Alias

We can also provide an alias to namespace or class while using it from other classes or PHP files to make it easier to write.

Example:

        use Examples as E;
        $example = new E\First();

Exercises

Create a new PHP File and:

  • Create a new class named Vehicle and:
    • Define properties model, color and price
    • Define a constructor that takes three parameters model, color and price for the class and set the model, color and price of the vehicle
    • Define a static method named displayInfo(), set its access to public to display the info about a vehicle.
    • Create an object and call the displayInfo() method to display the output.
  • Create a class named Car that inherits the Vehicle class and:
    • Define another static property isElectric and set its value to boolean true by default.
    • Override the constructor of Vehicle class to take fourth parameter isElectric and set it to the static property isElectric.
    • Create a non-static method isElectric() and return if the car is electric or not.
    • Create an object and call the displayInfo() for a car only if the car is electric.
  • Create a new class named Food in a new PHP file:
    • Set the namespace as Edible
    • Create a new property favourite.
    • Create a constructor to set your favourite food
    • Create a new method displayFood() and display the food you have set in the constructor.
    • Create an object of Food outside the class and call the displayFood() method.
    • Create another class in a new PHP file and name as JunkFood
    • Include the file with Food class
    • Use the Food class by using namespace.
    • Set the alias of Edible namespace as E.
    • Create an object using namespace and display the favourite food
