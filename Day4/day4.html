HTML Media

HTML Media

Multimedia files can also be embedded in a webpage. <video> and <audio> tags are used to embed videos and audios in a webpage respectively.

Embedding Videos in a Webpage 

          <video width="320" height="240" controls>
            <source src="sample.mp4" type="video/mp4">
            <source src="sample.ogg" type="video/ogg">
            Your browser does not support the video tag.
          </video>

Output

Explanation

In the example above, <video> tag is used to embed a video to the website.

Inside the video tag, multiple source tags can be used to specify the source of the video with different types. Browsers will embed the first recognized type. The text inside the video tag is only displayed if the video tag is not supported.

Attributes of Video Tag

  • width: to specify the width of the video
  • height: to specify the height of the video
  • controls: to display the controls of the video
  • autoplay: to autoplay the video
  • muted: to mute of the video
  • loop: to play the video in loop

Attributes of Source Tag

  • src: to specify the source of the video
  • type: to specify the type of the video; supported video formats: .mp4, .webm and .ogg.

Embedding Audios in a Webpage 

          <audio controls>
            <source src="sample.mpeg" type="audio/mpeg">
            <source src="sample.wav" type="audio/wav">
            Your browser does not support the audio tag.
          </audio>

Output

Explanation

In the example above, <audio> tag is used to embed a audio to the website.

Inside the audio tag, multiple source tags can be used to specify the source of the audio with different types. Browsers will embed the first recognized type. The text inside the audio tag is only displayed if the video tag is not supported.

Attributes of Audio Tag

  • controls: to display the controls of the audio
  • autoplay: to autoplay the audio
  • muted: to mute of the audio
  • loop: to play the audio in loop

Attributes of Source Tag

  • src: to specify the source of the audio
  • type: to specify the type of the audio; supported audio formats: .mpeg, .wav and .ogg.

HTML Semantics

HTML Semantics

With introduction of HTML5, various semantic elements have been introduced for creating the layout of a web document where non-semantic elements like table or div are still being commonly used for designing webpages.

List of HTML Semantic Elements

  1. article
  2. aside
  3. summary
  4. details
  5. figcaption
  6. figure
  7. footer
  8. header
  9. main
  10. nav
  11. section

Most of the semantic elements like header, footer, section, article, nav and aside act as container to other elements and doesn't have their own specific styling unless styled using CSS. However, it is easier to identify what contents that section contain using these semantics.

NB: The use of semantic elements is also helpful for SEO.

HTML Meta

HTML Meta

<meta> tag is used to provide more information about a webpage or a website. meta tags are always included inside head elements to specify character set, page description, keywords, author of the document and viewport settings.

Contents from meta tag are not displayed in the webpage but is used by browsers, search engines or other web services.

Attributes

  • charset
  • content
  • name

Usage of meta tags 

          <head>
            <meta name="charset" content="UTF-8">
            <meta name="description" content="Web Dev JRP | Day 4">
            <meta name="keywords" content="HTML, CSS, JavaScript">
            <meta name="author" content="Anil Nepal">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
          </head>

HTML Forms

HTML Forms

HTML Forms are used to collect information from the user. <form> tag is used for creating forms in HTML.

Attributes of HTML Forms

  • action: url for processing form data
  • method: get|post
  • target: _blank|_self
  • autocomplete: on|off
  • enctype: multipart/form-data|application/x-www-form-ulrencoded|text/plain

HTML Form Elements

  • input
  • label
  • select
  • option
  • optgroup
  • textarea
  • fieldset

Select element is used in HTML to display a list of options and let users select one or multiple options.

Example:
