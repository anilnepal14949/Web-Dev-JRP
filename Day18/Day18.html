About JavaScript

JavaScript is the most popular and widely used client-side scripting language i.e., scripts that run within a web browser.

Why use JavaScript?

Below are some of the purposes of using JavaScript in a webpage.

  • Modify content of webpage by adding or removing elements
  • Change the style and position of elements in a webpage
  • Create and attach events like mouse click, hover, keypress
  • Create alerts and confirm boxes for user interactivity
  • Perform operations based on user inputs and display the results
  • User validation before submitting to the server

Embedding JavaScript

Similar to CSS, JavaScript needs to be added to a webpage in order to make the scripts work.

Ways to embed JavaScript

  • Inline
  • Internal (Using script tag)
  • External

Adding Inline JavaScript

<button onclick="alert('Hello')"> Show Alert </button>

Adding Internal JavaScript

<script> document.write("I am displayed with JavaScript") </script>

Adding External JavaScript

<script src="script.js"></script>

JavaScript Rules

All external javascript files must have extension .js.

Comments in JavaScript

JavaScript supports single line as well as multi-line comments.

Single line comments begin with a double forward slash (//) while multi-line comments begin with a slash and an asterisk (/*) and ends with an asterisk followed by a slash (*/).

Case Sensitivity in JavaScript

Unlike HTML and CSS, JavaScript is a case-sensitive language i.e., variables hello, Hello and HELLO must be used in the same way it is declared.

Similarly, predefined methods like getElementById() cannot be written as getElementByID().

JavaScript Variables

Variables are used to store data like string, numbers, boolean, etc. Variables can be set, updated and retrieved whenever needed.

We can create variables using var keyword in javascript.

Example: var firstName = 'ABC'

Naming rules:

  • A variable name must start with a letter, an underscore (_) or a dollar sign ($)
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters (A-z, 0-9) and underscores
  • A variable name cannot contain spaces
  • A variable name cannot be a JavaScript keyword
NB: JavaScript variables are case sensitive

JavaScript Datatypes

Datatypes are used to specify what type of data can be stored and manipulated within a script.

The String Data Type

The string data type is used to represent textual data.

Example: var name = 'PQR'; or var name = "PQR";

The Number Data Type

The number data type is used to represent positive or negative numbers with or without decimal points.

Example: var a = 100; or var b = 10.25; or var c = -30;

The Boolean Data Type

The boolean data type is used to hold true or false values.

Example: var isTrue = true; or var isFalse = false;

The Undefined Data Type

The undefined data type is used to specify whether a variable is defined or undefined. Undefined means the variable has been declared, but has not been assigned a value.

Example: var name; alert(name);

In the example above, the output will be undefined as we are alerting name but that variable has no value set.

The Null Data Type

The null data type is used to declare a variable as null. Null refers to an empty string ("") for strings and a 0 for numbers.

Example: var name = null; or var count = null;

Apart from these datatypes, there are others too like objects, arrays and functions. We will cover all of these separately in coming sessions.

JavaScript Operators

Operators are used to perform some sort of actions.

Types of Operators:

  • Arithmetic Operators: +, -, *, /, %
  • Assignment Operators: =, +=, -=, /=, *=, %=
  • Concatenation Operators: +, +=
  • Increment/Decrement Operators: ++x, --x, x++, x--
  • Logical Operators: &&, ||, !
  • Comparison Operators: ==, ===, !=, !==, <, >, <=, >=

JavaScript Strings

A string is a sequence of letters, numbers, special characters and arithmetic values or combination of all.

Escape Character (\):

An escape character is used to escape a single quote or a double quote.

Example: var str1 = "I am tring to \"escape\" character";

Escape Sequences :

Escape sequence are useful when we want to use characters that can't be typed using a keyboard.

List of some of the mostly used escape sequences:

  • \n: Used to insert a newline
  • \t: Used to insert a tab

String Operations

JavaScript provides several properties and methods to perform operations on string values.

Getting the length of a String

The length property is used to return the length of a string; also includes spaces, new lines(\n) and tab(\t).

Example: var para = "This is a paragraph!"; alert(para.length);

Finding a String inside another String

The indexOf() and lastIndexOf() methods are used to find a substring within another string.

Example: var para = "Do you think I can find this in this para?"; alert(para.indexOf("this")); alert(para.lastIndexOf("this"));

Extracting a Substring from a String

The slice() method is used to extract a part or substring from a string.

The slice() method accepts two parameters; startIndex, endIndex.

Example: var para = "Let's take some portion of this paragraph"; alert(para.slice(10, 15));

The substring() method is also used to extract substring from a string.

Similarly, JavaScript also provides substr() method that accepts two parameters startIndex and length.

Example: var para = "Let's take some portion of this paragraph"; alert(para.substr(10, 15));

Replacing the contents of a String

The replace() method can be used to replace part of a string with another string.

Example: var para = "Let's replace this."; alert(para.replace("this", "that"));

Converting a String to uppercase or lowercase

The toUpperCase() and toLowerCase() methods are used to convert a string to uppercase and lowercase respectively.

Example: var name="my name"; alert(name.toUpperCase()); alert(name.toLowerCase());

Concatenating Strings

We can use concatenation operators (+, +=) for concatenating strings in JavaScript.

Example: var firstName = "AB"; var lastName = "C"; var name = firstName + lastName;

Splitting Strings

The split() method can be used to split a string into an array of strings.

Example: var fruits = "Apple, Banana, Mango, Grapes"; var fruitsArr = fruitsList.split(", ");

JavaScript Numbers

JavaScript supports both integer and floating-point numbers.

Example: var x = 10; var x = 3.15; var x = 0xff;

Number Operations

JavaScript provides several properties and methods to perform operations on number values.

Parsing Integers from Strings:

The parseInt() method can be used to parse an integer from a string.

Example: console.log(parseInt("3.15")); console.log(parseInt("50px"));

The parseFloat() method can be used to parse floating-point number from a string.

Example: console.log(parseFloat("3.15")); console.log(parseFloat("124.5 lbs"));

Converting Numbers to Strings

The toString() method can be used to convert a number to its string equivalent.

Example: var x = 12; console.log(x.toString());

NB: We will learn more math operations in coming sessions.

JavaScript Conditional Statements

JavaScript allows to perform actions based on the results of a logical or comparative test conditions at run time.

List of conditional statements:

  • The if statement
  • The if...else statement
  • The if...elseif...else statement
  • The switch...case statement

The if statement

The if statement is used to execute a block of code only if the specified condition is true.

Syntax:

if(condition) { // blocks of code to be executed }

The if...else statement

The if...else statement is used to execute a block of code only if the specified condition is true and another block of code if it is false.

Syntax:

if(condition) { // blocks of code to be executed if condition is true } else { // blocks of code to be executed if condition is false }

Ternary Operator

The ternary operator provides the shorthand way of writing the if...else statement.

Syntax: var result = (condition) ? value1 : value2;

The if...elseif...else statement

The if...elseif...else statement is used to multiple if...else statements.

Syntax:

if(condition1) { // blocks of code to be executed if condition1 is true } else if(condition2) { // blocks of code to be executed if condition1 is false and condition2 is true } else { // blocks of code to be executed if both condition1 and condition2 are false }

The switch...case statement

This statement is an alternative to the if...elseif...else statement.

Syntax:

switch(expression) { case value1: { // blocks of code to be executed if x === value1 break; } case value2: { // blocks of code to be executed if x === value2 break; } ... default: // blocks of code to be executed if x is different from all values
