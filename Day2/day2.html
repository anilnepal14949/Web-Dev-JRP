HTML Tags, Elements & Attributes

HTML Tags

HTML tags begin with < symbol and end with > symbol.

Example: <b></b> | <p></p>

HTML Elements

HTML elements contain the start tag, content and an end tag.

Example: <b> This is the bold text. </b> | <p> This is a paragraph tag.</p>

HTML Attributes

HTML attributes provide additional properties to tags.

Example: <p align="center"> This is a center aligned text. </p>

In the example above, align is an attribute of p tag and center written in double quotes("") is the value of the attribute.

While using nested elements, the closing of the tags need to in the order of tags' opening.

Example: <p><b> This is a bold text in paragraph tag. </b></p> Here, the nested tag <b> is closed first before the parent tag <p>.

List of basic tags with their attributes:

  • <body>...</body>: bgcolor, background, text, link, alink, vlink
  • <p>...</p>: align
  • <font>...</font>: size, face, color

Common attributes:

These attributes can be used with any tag in HTML

  • id
  • class
  • style
  • title

Formatting HTML

Bold Text

<b>...</b> and <strong>...</strong> tags are used to make the text bold.

Italic Text

<i>...</i> and <em>...</em> tags are used to italicize the text.

Underline Text

<u>...</u> tag is used to underline the text.

Strike Text

<strike>...</strike> tag is used to strike through the text.

Superscript Text

<sup>...</sup> tag is used to make the text superscript. Eg: a2+b2

Subscript Text

<sub>...</sub> tag is used to make the text subscript. Eg: H20

Larger Text

<big>...</big> tag is used to make the text large.

Smaller Text

<small>...</small> tag is used to make the text small.

Preformatted Text

<pre>...</pre> tag is used to make the text 

        preformatted.

Marked Text

<mark>...</mark> tag is used to highlight the text.

Quoting Text

<blockquote>...</blockquote> tag is used to quote the text. Eg:

This is a blockquote

Commented Text

Comments in HTML are written within <!-- ..... --> tags and are ignored by web browser. Eg: <-- This is a comment. -->

HTML Images

HTML Images

<img> tag is an empty tag used to insert image in a webpage.

<img> Attributes

All global attributes(id, class, title, style) are supported by <img> tag. Other than the global attributes, <img> tag has its own set of attributes listed below:

  • src
  • width
  • height
  • alt
  • align
  • border

Usage

<img src="tech.jpg"> will display image below:

<img src="tech.jpg" width="100"> will set the image width to 100 and height auto:

<img src="tech.jpg" height="100"> will set the image height to 100 and width auto :

<img src="tech.jpg" width="100" height="100"> will set the width and height to 100:

<img src="tech.jpg" height="100" align="left"> will set the height of the image to 100 and align the image to left:





<img src="tech.jpg" height="100" border="5"> will set the image height to 100 and give the border of size 5 to the image:

<img src="tech.jpg" alt="Tech Image"> will give an alternative title to the image if the image cannot be loaded for some reasons:

HTML Tables

HTML Tables are used to arrange data in tabular form (rows and columns).

Simple HTML Table 

        
          <!DOCTYPE html> 
          <html>
          <head>
            <title>Sample Table</title>
          </head>
          <body>
            <table border="1">
              <tr>
                <th>
                  Name
                </th>
                <th>
                  Class
                </th>
              </tr>
              <tr>
                <td>
                  ABC
                </td>
                <td>
                  Class 1
                </td>
              </tr>
              <tr>
                <td>
                  PQR
                </td>
                <td>
                  Class 5
                </td>
              </tr>
            </table>
          </body>
          </html>

Output:

Name Class
ABC Class 1
PQR Class 5

List of Attributes

  • border
  • cellpadding
  • cellspacing
  • width
  • height
  • align (left, right, center)
  • valign (top, middle, bottom)
  • bgcolor (deprecated)
  • colspan (used with td/th)
  • rowspan (used with td/th)
